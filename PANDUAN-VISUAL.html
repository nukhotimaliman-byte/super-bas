üîçTroubleshooting

‚ùå Website blank / Error 404 saat refresh

Solusi:

  • Clear cache browser (Ctrl+F5 atau Cmd+Shift+R)
  • Pastikan .htaccess ada di public_html
  • Tunggu 5 menit agar cache server terupdate
  • Hubungi Rumahweb: minta aktifkan mod_rewrite jika masih error

‚ùå File tidak ter-upload

Solusi:

  • Coba upload 1 file dulu untuk test koneksi
  • Gunakan FTP jika File Manager timeout
  • Check ukuran file (biasanya max 100MB per file)
  • Coba di browser lain jika masih gagal

‚ùå Halaman / CSS / JavaScript tidak muncul

Solusi:

  • Hard refresh (Ctrl+F5)
  • Check folder assets ada atau tidak di public_html
  • Buka DevTools (F12) untuk lihat error di console
  • Pastikan URL benar (jangan ada typo di subdomain)
